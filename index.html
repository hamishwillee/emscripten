Compile native code to the web

Emscripten takes LLVM bytecode generated from C/C++ and compiles it into asm.js (a highly optimisable strict subset of JavaScript). This can be run on the web without plugins.

Native code

Compile C and C++ code into LLVM using Clang. Emscripten converts the LLVM to JavaScript.

Runtimes

Python and other interpreted languages can be run on the web by compiling their native runtimes to JavaScript

Fast

Better than half native speed is currently achievable on some browsers/benchmarks.
